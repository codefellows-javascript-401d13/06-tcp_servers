# ${1:TCP SERVER}
This is a tcp messenger app. You can connect and share messages with friends or privately.
## Installation
To install open Fork this repository
Clone this repository to your machine
To install dependencies run npm i
Run node server.js to start the server
To connect to the server run "telnet ip-address port"
ex: "telnet 123.456.7.89 8000"
## Usage
@nickname newname - sets your name on the server
@all msg - sends message to  
@dm username message - sends a message to that user
